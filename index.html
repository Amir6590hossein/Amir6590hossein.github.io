Exercise 1: Fill in the Blanks

Differential Equations

Each individual solution of a differential equation is called a particular solution. A general solution of an ordinary differential equation is a nonempty set of solutions specified by an expression which contains at least one parameter and which becomes a representation of a particular solution when its parameters are all replaced by numbers. A solution is said to be relative to any given general solution if it does not belong to that solution set.

Question 6

Translate the following text into Farsi:

An expression of the form c1y1 + c2y2 + ... + cmym, in which c1, c2, ..., cm are arbitrary constants, is called a linear combination of variables y1, y2, ..., ym because each term in the expression is linear, i.e., homogeneous of degree 1, in these m variables. One extremely important property of every homogeneous linear equation is that, if y1 and y2 are any particular solutions, then, for all values of the constants c1 and c2, the linear combination c1y1 + c2y2 is also a solution.

(ترجمه فارسی: عبارت‌هایی به شکل c1y1 + c2y2 + ... + cmym که در آن c1, c2, ..., cm ثابت‌های دلخواه هستند، ترکیب خطی متغیرهای y1, y2, ..., ym نامیده می‌شوند، زیرا هر جمله در این عبارت خطی است، یعنی همگن از درجه ۱ در این m متغیر. یکی از ویژگی‌های بسیار مهم هر معادله خطی همگن این است که اگر y1 و y2 هر دو جواب خاص باشند، آنگاه برای همه مقادیر ثابت‌های c1 و c2، ترکیب خطی c1y1 + c2y2 نیز یک جواب است.)
